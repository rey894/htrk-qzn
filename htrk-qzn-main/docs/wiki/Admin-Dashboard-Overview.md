# Admin Dashboard Overview

A comprehensive overview of the Quezon Municipal Website Admin Dashboard.

## What is the Admin Dashboard?

The Admin Dashboard is a WordPress-like content management system that allows authorized personnel to manage the municipal website content without requiring technical knowledge.

## Accessing the Dashboard

1. Navigate to `https://quezonbukidnon.com/auth`
2. Enter your admin credentials
3. Click "Sign In"
4. You'll be redirected to the Admin Dashboard

## Dashboard Sections

### Main Menu

The sidebar menu contains the following sections:

1. **Dashboard** - Overview and statistics
2. **News** - Manage news articles
3. **Events** - Manage calendar events
4. **Documents** - Manage public documents
5. **Media** - Media library (images and files)
6. **Navigation** - Manage website navigation menus
7. **Messages** - View contact form messages
8. **Users** - Manage user accounts (Admin only)
9. **Settings** - Site configuration (Admin only)
10. **BAC Documents** - Manage procurement documents (BAC role only)

### Dashboard Statistics

The main dashboard shows:

- **Total News**: Count of published and draft news articles
- **Total Events**: Count of published and draft events
- **Total Documents**: Count of published and draft documents
- **Unread Messages**: Count of pending contact form messages

### Role-Based Access

Different roles have different access levels:

- **Admin**: Full access to all sections
- **Moderator**: Can edit content (news, events, documents)
- **User**: Basic access (limited features)
- **BAC**: Access to BAC Documents only

## Quick Actions

### Creating Content

Most sections have a "Create New" or "Add" button:

1. Click the section in the sidebar (e.g., "News")
2. Click "Create New" or "Add [Item]" button
3. Fill in the form
4. Click "Save" or "Publish"

### Editing Content

1. Navigate to the section
2. Find the item you want to edit
3. Click the Edit icon (pencil)
4. Make changes
5. Click "Save"

### Deleting Content

1. Navigate to the section
2. Find the item you want to delete
3. Click the Delete icon (trash)
4. Confirm deletion

**Warning**: Deletion is permanent and cannot be undone!

## Best Practices

### Content Management

- **Save as Draft** first, then publish when ready
- **Add Alt Text** to images for accessibility
- **Use clear titles** for better SEO
- **Categorize content** for organization

### Security

- **Log out when finished** managing content
- **Don't share admin credentials**
- **Use strong passwords**
- **Review changes** before publishing

### File Management

- **Optimize images** before uploading (max 10MB)
- **Use descriptive file names**
- **Organize media** in the Media Library
- **Delete unused files** to save storage

## Navigation Tips

### Sidebar Menu

- **Collapsible Sections**: Some sections may have sub-menus
- **Active Indicator**: Current section is highlighted
- **Quick Access**: Click section name to expand/collapse

### Dashboard Layout

- **Top Bar**: Shows current user and logout button
- **Sidebar**: Navigation menu (always visible)
- **Main Area**: Content area (changes based on section)
- **Footer**: System information and links
