# Getting Started

Welcome to the Quezon Municipal Website! This guide will help you get started with managing the website content.

## First Steps

### 1. Accessing the Admin Dashboard

1. Navigate to the admin login page: `https://quezonbukidnon.com/auth`
2. Enter your admin credentials (provided by your system administrator)
3. Click "Sign In"
4. You'll be redirected to the Admin Dashboard

### 2. Understanding Your Role

The website uses role-based access control:

- **Admin**: Full access to all content management features
- **Moderator**: Can edit content (news, events, documents)
- **User**: Basic access (limited features)
- **BAC**: Access to BAC (procurement) documents only

If you need different permissions, contact your system administrator.

### 3. Dashboard Overview

The Admin Dashboard provides:

- **Statistics**: Quick overview of content counts (news, events, documents)
- **Navigation Menu**: Sidebar menu for accessing different sections
- **Recent Activity**: Recent changes and updates

## Common Tasks

### Publishing a News Article

1. Click **"News"** in the sidebar menu
2. Click **"Create New"** or **"Add News"** button
3. Fill in the form:
   - **Title**: Article headline
   - **Content**: Full article text
   - **Excerpt**: Brief summary (1-2 sentences)
   - **Featured Image**: Upload or use image from Media Library
   - **Category**: Select appropriate category
   - **Status**: Choose "Draft" or "Published"
4. Click **"Save"** or **"Publish"**

### Adding an Event

1. Click **"Events"** in the sidebar menu
2. Click **"Create New"** or **"Add Event"** button
3. Enter event details:
   - **Title**: Event name
   - **Description**: Event details
   - **Event Date**: When it starts
   - **End Date**: When it ends (optional)
   - **Location**: Where it's happening
   - **Category**: Type of event
   - **Status**: Upcoming, Ongoing, or Past
4. Click **"Save"**

### Uploading a Document

1. Click **"Documents"** in the sidebar menu
2. Click **"Create New"** or **"Upload Document"** button
3. Fill in:
   - **Title**: Document name
   - **Description**: What the document is about
   - **Category**: Type of document
   - **Department**: Which office it belongs to
   - **File**: Upload PDF or other files (max 5MB)
   - **Status**: Choose "Published" to make it visible
4. Click **"Save"**

## Best Practices

### Content Guidelines

- **Always add Alt Text** to images for accessibility and SEO
- **Use clear, descriptive titles** for better search engine visibility
- **Keep excerpts brief** (1-2 sentences) for better listings
- **Categorize content** for better organization and navigation
- **Save as Draft first**, then publish when ready for review

### File Upload Guidelines

- **Images**: Maximum 10MB
- **Documents**: Maximum 5MB
- **Supported formats**: Images (JPG, PNG, WebP), PDFs, Office documents
- **Always optimize images** before uploading for faster page loads

### Security Tips

- **Use strong passwords** and change them regularly
- **Don't share admin credentials** with others
- **Log out when finished** managing content
- **Review changes** before publishing
